# Modern React Calculator

A feature-rich, scientific calculator application built with React and Vite. It features a modern, responsive user interface with professional gradient aesthetics and glassmorphism effects.

## Features

- **Standard Operations:** Addition, subtraction, multiplication, division.
- **Scientific Functions:** Square root (√), Square (x²), Power (x^y), Natural Log (ln), Log base 10 (log), Percentage (%), Pi (π).
- **Modern UI:** Professional deep blue gradient background with glassmorphism effects.
- **Responsive Design:** Fully responsive layout that adapts seamlessly to desktop, tablet, and mobile devices.
- **Keyboard Support:** Full keyboard support for digits and operators.

## Tech Stack

- **Framework:** React 18
- **Build Tool:** Vite
- **Styling:** CSS3 (Grid, Flexbox, CSS Variables, Gradients, Media Queries)

## Getting Started

### Prerequisites

- Node.js installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```bash
   cd Calculator
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:5173` (or the URL shown in your terminal).

## Building for Production

To create a production build:

```bash
npm run build
```

This will generate optimal static assets in the `dist` directory.
